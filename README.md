---
No terminal:
---

### Clonar o repositótio

```sh
$ git clone https://github.com/FernandoSouzaMarques/webpack-default.git
```


### Instalação

```sh
$ yarn install
```

### Desenvolvimento

```sh
$ yarn start
```

### Compilar o html

```sh
$ yarn build
```
